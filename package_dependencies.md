# Required Dependencies for Bidirectional Export

Add these dependencies to your package.json:

```json
{
  "dependencies": {
    "jspdf": "^2.5.1",
    "date-fns": "^2.30.0"
  },
  "devDependencies": {
    "@types/jspdf": "^2.3.0"
  }
}
```

Install with:
```bash
npm install jspdf date-fns
npm install --save-dev @types/jspdf
```
